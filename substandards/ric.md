# UAID x RIC - Refinitiv Identification Codes

RIC defines a ticker-like code used by Refinitiv to identify financial instruments and indices in Refinitiv financial information networks.

https://en.wikipedia.org/wiki/Refinitiv_Identification_Code

### Examples

| UAID | What it is |
| ------------ | ------------ |
| **`uaid:ric:ibm.n`** | Stock shares of IBM Corp. traded on the New York Stock Exchange |
| **`uaid:ric:ibm.l`** | Stock shares of IBM Corp. traded on the London Stock Exchange |
| **`uaid:ric:msft.oq`** | Stock shares of Microsoft Corp. traded on the NASDAQ |
| **`uaid:ric:wmt.n`** | Stock shares of Walmart Inc. traded on the New York Stock Exchange |
| **`uaid:ric:.dji`** | The Dow Jones Industrial Average |
| **`uaid:ric:.spx`** | The S&P 500 Index |
