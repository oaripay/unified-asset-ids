# UAID x ISO 4217 - Currency Codes

ISO 4217 defines alpha codes and numeric codes for the representation of currencies, funds and precious metals, and provides information about their minor units.

https://en.wikipedia.org/wiki/ISO_4217

### Examples

| UAID | What it is |
| ------------ | ------------ |
| **`uaid:iso4217:usd`** | The U.S. Dollar |
| **`uaid:iso4217:eur`** | The Euro |
| **`uaid:iso4217:jpy`** | The Japanese Yen |
| **`uaid:iso4217:gbp`** | Pound Sterling |
| **`uaid:iso4217:xau`** | Gold, defined as one troy ounce |
| **`uaid:iso4217:xag`** | Silver, defined as one troy ounce |
