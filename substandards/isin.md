# UAID x ISIN - International Securities Identification Numbers

ISIN defines a way to uniquely identify securities globally with a 12-character alphanumeric code for use in clearing, reporting and settlement of trades.

https://en.wikipedia.org/wiki/International_Securities_Identification_Number

### Examples

| UAID | What it is |
| ------------ | ------------ |
| **`uaid:isin:us0231351067`** | Stock shares of Amazon.com, Inc. |
| **`uaid:isin:us0378331005`** | Stock shares of Apple Inc. |
| **`uaid:isin:us5949181045`** | Stock shares of Microsoft Corp. |
| **`uaid:isin:gb0002634946`** | Stock shares of BAE Systems plc |
| **`uaid:isin:au0000xvgza3`** | Treasury Corporation of Victoria 5 3/4% 2005-2016 |
